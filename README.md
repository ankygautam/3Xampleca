# 3xample.ca

A modern portfolio and showcase website for **3xample.ca**, built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**.

This project serves as a public hub for showcasing experiments, products, and build-in-public ideas around **AI-assisted**, **agentic**, and **modern web applications**. The site is designed as a clean single-page experience with centralized content configuration, simple maintenance, and static export support for easy deployment.

---

## Overview

**3xample.ca** is positioned as a platform for turning ideas into polished digital products.

The site highlights:
- public experiments
- AI-assisted product builds
- featured project showcases
- upcoming tools and concepts
- a build-in-public development direction

The repository is structured as a lightweight frontend application using the **Next.js App Router**, with core homepage content managed through centralized configuration files.

## Tech Stack

- **Framework:** Next.js 15
- **UI Library:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Architecture:** App Router
- **Deployment Style:** Static export
- **Publishing:** GitHub Pages compatible workflow via `gh-pages`

## Features

- Clean single-page portfolio/showcase layout
- Centralized content management
- Reusable UI components
- Static export support for simple deployment

- Lightweight structure for easy future scaling
- Ideal base for portfolio, startup showcase, or AI product launch page
