# 3xample

Single-page Next.js site for [3xample.ca](https://3xample.ca), a build-in-public showcase for AI-agentic web applications.

## Local development

```bash
npm install
npm run dev
```

## Notes

- Built with Next.js App Router and Tailwind CSS
- Static-friendly and deployable to Vercel with zero extra configuration
- All homepage content is driven by `config.ts` and `projects.ts`
